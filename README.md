# WebDevTutorial
This repo is for beginners who want to learn web dev.
There are 3 main files in this project :
1.html
2.css
3.JS

Html file shows how to use html tags and what are the main tags present in html.

CSS file shows how to style a webpage and all the css components are embedded in this page.

JS file gives a glimpse of all the syntax, functions, varaibles and all related to javascript.
